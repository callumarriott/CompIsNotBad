Central Processing Unit

A central processing unit (CPU) is the electronic circuitry within a computer that carries out the instructions of a computer program by performing the basic arithmetic, logic, controlling and input/output (I/O) operations specified by the instructions. CPU has been started to use within the computer industry at leasy since 1960s. Even though it has been 60 years since CPU first came out, the fundermental structure and operation of a CPU remains almost unchanged.
